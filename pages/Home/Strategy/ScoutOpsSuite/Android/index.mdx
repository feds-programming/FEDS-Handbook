## About Scout-Ops-Android

Scout-Ops-Android is a mobile-friendly scouting application designed for FRC matches. Originally developed by Team 201 (FEDS), the app provides a comprehensive solution for match scouting, data collection, and sharing during competitions.

<Callout type="info">
  **Latest Version:** 4.0 (released March 27, 2025)
</Callout>

## Key Features

### User-Friendly Interface
- Intuitive match flow that follows the natural progression of FRC matches
- Responsive design that works on various screen sizes
- Light and dark themes for different lighting conditions

### Robust Data Collection
- Autonomous phase tracking
- Teleop scoring metrics
- Endgame performance recording
- Custom comments for qualitative observations

### Flexible Data Transfer
- QR code generation for quick scanning
- Bluetooth PAN connectivity for wireless transfers
- Local storage for offline operation

### Offline Functionality
- Download event data before competition
- No internet required during operation
- Secure local storage of all scouting data

## System Requirements

- Android 7.0 or newer
- 50 MB of available storage space
- Bluetooth capability (for PAN transfers)
- Camera (for QR code scanning)

## Getting Started

<Steps>
### Install the app
Download Scout-Ops-Android from the Amazon App Store

### Configure your settings
Set up your scouter name and grant required permissions

### Load event data
Enter the event key to download match schedules

### Start scouting!
Begin collecting match data with a tap on "Start Match"
</Steps>
