# Subsystem Overview

<div></div>
### What is a Subsystem?

An FRC robot is made from multiple <u>Subsystems</u>, or parts of a robot that each perform a task separately and while working together.
<div></div>

<Callout type="warning" emoji={"âš "}>FEDS201 has created a Custom Framework, its mildly different from normal WpiLib and FRC Setup </Callout>

### How do we program subsystems?

We use Java object-oriented programming to program our robot. We create subsystems using a class and give it functions using commands.
