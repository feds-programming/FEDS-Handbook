### Tools and Setup

Welcome to the Programming subgroup! This guide is designed to help you get started with the tools and technologies used by our team. Whether you're new to programming or looking to sharpen your skills, this guide has something for everyone. We'll cover the software, hardware, and libraries you need to develop the code that controls the robot.
