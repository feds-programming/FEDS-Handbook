Welcome to the FEDS201 Components Library! This page demonstrates the power of our parameter-driven, generic components that can create virtually any widget or UI element needed across the handbook.

## 🎯 Component Philosophy

Our library follows these principles:
- **Parameter-driven**: Every component is highly configurable through props
- **Responsive**: Works beautifully across all screen sizes
- **Themeable**: Adapts to light and dark themes automatically
- **Animated**: Smooth, professional animations that enhance UX
- **Reusable**: One component, infinite possibilities

## 🚀 Getting Started

To use these components in your pages, simply import them from the Components library:

```jsx
import { 
  GenericButton, 
  GenericCard, 
  GenericHero, 
  GenericStats,
  GenericGrid,
  GenericTimeline,
  GenericWidget,
  GenericNavigation 
} from '../Components'

// Use with custom props
<GenericCard
  variant="modern"
  size="lg"
  interactive={true}
  customStyles={{ background: 'your-custom-gradient' }}
>
  Your content here
</GenericCard>
```

### Pre-configured Components

For common use cases, we also provide pre-configured components:

```jsx
import { 
  ProgrammingHero,
  StatsSection,
  ModernCard,
  ResourceWidget 
} from '../Components'

// These are the generic components with predefined props
<ProgrammingHero />
<StatsSection />
```
