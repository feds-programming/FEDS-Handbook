# General Rules:
1. **Be Consistent**: Use the same style and formatting throughout the guide.
2. **Be Concise**: Keep your writing clear and to the point.
3. **Be Informative**: Provide useful information that helps the reader.

# Page Structure:
1. **Title**: The title should be descriptive and concise.
2. **Import Statements**: Import any components or icons you need for the page.
3. **Introduction**: Provide an overview of the topic and what the reader can expect to learn.
4. **Content**: Organize the content into sections with headings and subheadings.
5. **Links**: Include links to relevant resources or external websites.

# Programming Rules:
1. No-spaces in file names.
2. Add your file name to the _meta.json file.
```js
export default {
  foo: '',
  1992_10_21: '',
  1: ''
}
```
3. Use the same style and formatting throughout the guide.
